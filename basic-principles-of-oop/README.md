## 📚 Принципы ООП в проекте

### 1. **Инкапсуляция** (Encapsulation)
- Все атрибуты классов защищены (префикс `_`)
- Доступ к данным осуществляется через публичные методы
- Пример:
  ```python
  self._name = name  # защищенный атрибут
  def get_name(self):  # публичный метод доступа
      return self._name
  ```

### 2. **Наследование** (Inheritance)
- `Animal` ← `Lion` ← `Lioness`
- Переиспользование кода родительских классов
- Дополнение и специализация функционала

### 3. **Полиморфизм** (Polymorphism)
- Переопределение методов `make_sound()`, `eat()`, `__str__()`
- Единый интерфейс для разных типов животных
- Пример:
  ```python
  # В Animal
  def make_sound(self):
      raise NotImplementedError
  
  # В Lion
  def make_sound(self):
      print(f"{self._name} рычит!")
  ```

## 🦁 Классовая иерархия

```
        Animal (базовый класс)
            │
            ▼
          Lion (лев)
            │
            ▼
        Lioness (львица)
```

### Детали классов:

**Animal** (абстрактная сущность):
- Атрибуты: имя, вид, возраст, голод
- Методы: базовые действия животного

**Lion** (специализация):
- Уникальные атрибуты: размер гривы, статус вожака
- Уникальные методы: охота, стать вожаком
- Переопределенные методы: питание (только мясо)

**Lioness** (дальнейшая специализация):
- Уникальные атрибуты: количество детенышей
- Уникальные методы: рождение детенышей
- Особенность: отсутствие гривы

## 🚀 Пример использования

```python
# Создание объектов
male_lion = Lion(name="Гриша", age=5, mane_size="Большая")
female_lion = Lioness(name="Света", age=4, cubs_count=2)

# Использование методов
male_lion.make_sound()  # Полиморфизм
male_lion.hunt("антилопа")
female_lion.give_birth(1)

# Доступ к данным через геттеры (инкапсуляция)
print(f"Имя льва: {male_lion.get_name()}")
print(f"Количество детенышей: {female_lion.get_cubs_count()}")
```
